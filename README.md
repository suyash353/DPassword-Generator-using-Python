# Password-Generator-using-Python

## How to Use?

## Steps :

### 1) Clone Repository 
        git clone http://github.com/ 

### 2) Create a virtual environment 
        conda create venv python==3.8 -y  

### 3) Activate the virtual environment 
        conda activate ./venv 

### 4) Run this command 
        pip install -r requirements.txt
        
 ### 5) Run this command 
        python app.py
